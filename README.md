# 🗳️ Blockchain-Based Secure Voting System

A secure and transparent voting system that simulates blockchain technology using modern web technologies and Firebase. This project ensures vote integrity, user authentication, and fraud prevention.

---

## 🚀 Features

* 🔐 Secure Authentication (Email / Aadhaar simulation)
* 🧑‍💼 Unique Voter ID for each user
* 🗳️ One-person-one-vote system
* ⛓️ Blockchain-like vote storage (immutable records)
* 📊 Real-time voting results
* 🚨 Fraud detection system
* 📱 User-friendly interface
* 🎤 (Optional) Voice-assisted voting

---

## 🛠️ Tech Stack

* Frontend: React / Next.js
* Backend: Firebase / Node.js
* Database: Firestore / Drizzle ORM
* Validation: Zod
* Package Manager: pnpm

# 📥 1. Clone Project from GitHub (Recommended)

## Step 1: Copy repo link

Go to your repository and copy the HTTPS or SSH URL.

## Step 2: Open terminal in VS Code

```bash
git clone https://github.com/harshagv25/Voting-system-using-blockchain.git
```

## Step 3: Open project

```bash
cd Voting-system-using-blockchain
code .
```

---

# 📦 2. Install Dependencies

Make sure you have Node.js installed.

Then run:

```bash
npm install
```

OR (if using pnpm):

```bash
npm install -g pnpm
pnpm install
```

---

# ▶️ 3. Run the Project

```bash
npm run dev
```

OR

```bash
pnpm dev
```

👉 Open browser:

```
http://localhost:3000
```

---

# 📁 4. Run Project Using ZIP File (No Git Required)

## Step 1: Download ZIP

* Go to GitHub repo

## Step 2: Extract ZIP

* Right-click → Extract All

## Step 3: Open in VS Code

## Step 4: Install dependencies


OR


## Step 5: Run project

```bash

```
